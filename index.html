7.1 THE INTERNET

7.1.1 Definition

The Internet is a very big network of computers from all around the world, and they're all connected to each other using something called the TCP/IP protocol. This huge network lets us do many things.

7.1 THE INTERNET

7.1.2 Use of the internet

- Get information

- Communication

- Online shopping

- Education

- Entertainment

- Working remotely

- Access public services

- Social connection

- Doing researches

- Solving problems

7.1 THE INTERNET

7.1.3 Connection

- An Internet connection device: like a cable modem, DSL, or a router. These days, most use Wi-Fi devices to connect wirelessly to the Internet.

- Physical connections: Connect the Internet device to your computer using an Ethernet cable or doing so wirelessly.

To connect to the Internet, follow these steps:

Setup: Configure with the right settings on your device, including Wi-Fi name and password (if used) and the login details provided by the ISP.

Test the connection: After finishing setting up, check the connection by opening a web browser and accessing a website. If you can successfully access it, you're connected to the Internet.

Internet service: Sign up with an Internet service provider (ISP), like a telecom company or a fiber optic service, one example is Viettel.

7.1 THE INTERNET

7.1.4 Web Browser definition

A web browser is a software used to access and browse websites on the Internet. It allows for interaction between the user and web documents, allowing users to interact with Internet contents easily.

7.1 THE INTERNET

7.1.5 Online Services

- E-mail (Electronic Mail): allows users to read, reply to, forward, or delete the email if you want. Messages in emails can have several types of information like news, files, pictures, videos, and other files.

- FTP/HTTP (File Transfer Protocol): a way to move files on the internet. It helps the user remotely upload and download files to a server. To use FTP, the user needs to use an FTP application and sign in. FTP us used to to share big files and manage them from afar.

- Telnet/SSH: allows user to connect to work with remote computers as if they were there with that computer. By using a Telnet application, users can do things like logging in, giving commands, and accessing computer software and files from far away.

For example: controlling cameras, home smart appliances, machines in factories, …

7.1 THE INTERNET

7.1.6 Electronic mail (Email)

- Email: An electronic communication service to send and receive messages. It's replaced traditional methods of communication like letters and faxes.

- Email address: contains a username and a domain, like example@gmail.com. The user can check their inbox using services like Gmail, Outlook, Proton mail and more.

- Email allows for quick message delivery, easy communication, information storage, and at the same time, security through spam and harmful mail blocking and filtering.

7.1 THE INTERNET

7.1.7 World Wide Web (W.W.W)

- An information system on the Internet which allows the user to access and search for information from different websites. It was developed by Tim Berners-Lee in 1989 and became an important part of the technology revolution. It’s seen as a web of connected documents through “hyperlinks”.

- The WWW structure includes web pages, hyperlinks, and web browsers. Each web page is an electronic document written in markup languages like HTML, contains information, pictures, videos, or audios. It’s stored in a web server and accessible through URL addresses.

-The WWW has changed how we access information and communicate globally. It's opened the door to sharing knowledge, connecting with people and created a diverse online environment.

7.2 MULTIMEDIA

7.2.1 Definition

A way of using several different media of communication to deliver information. Multimedia is a combination of audios, pictures, videos, texts, and interactive elements to create a multi-dimensional experience to the user.

7.2 MULTIMEDIA

7.2.2 Popular

- Radio and TV

- Print media

- Internet

- Social media

7.2 MULTIMEDIA

7.2.3 Application

- Education: Multimedia is used widely in education to provide engaging and interactive learning materials to students. A few applications are teaching videos, online lessons, education software and multimedia materials.

- Entertainment: Multimedia an important part of the entertainment industry. It includes applications like movies, TV programs, video games, music and platforms like Netflix, Spotify, Youtube and Odysee

- Advertising and marketing: Companies use multimedia forms like TV advertisement, online advertisement videos, social media advertisement, etc to reach customers.

- Social media: Multimedia is used to allow the user to share information, pictures and videos and interact with each other. This creates an online environment to connect and interact.

- Business communication: Companies use multimedia to convey their information, products and services. They use tools like introduction videos, news, interactive websites, etc to communicate with customers and business partners.

7.3 INTERNET VS ETHERNET

- Internet and Ethernet are two concepts both about network connection, but they are very different. The Internet is a worldwide connection system that lets devices access resources and services from all over the world. Ethernet, on the other hand, is a wired technology for sending data between devices in a local network.

Some main differences between Internet and Ethernet are:

Internet Ethernet
Purpose Connect, share information globally (public) Connect, share data locally (private)
Scope No geographic limits, as long as there's a connection to ISP (TCP) Small scope: inside an office, classroom, school, or building.
Speed Affected by many elements and maximally 1GB/s More stable, 10Mb/s-100Gb/s
Operation Packet switching (split packets of data and send them through various routes, then put back together in order at the destination) Bit transmission (information flows continuously, one bit at a time)
Main Protocols TCP/IP CSMA/CD or CSMA/CA
